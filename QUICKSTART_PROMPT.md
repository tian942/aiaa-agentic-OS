```

1. First, clone the repository:
```

## What Happens Next

After you paste that prompt, Claude Code will:

1. **Download the system** from GitHub
2. **Install all dependencies** (Python packages)
3. **Launch the setup wizard** which walks you through:
   - Getting your API key (takes 2 minutes, free)
   - Setting up your agency profile
   - Adding your first client
   - Running your first workflow

Just follow along and answer the questions. Claude Code handles the rest.
