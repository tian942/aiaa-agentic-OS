## Pre-Kickoff (Complete before project start)

### Access & Credentials
- [ ] Client provides admin access to relevant systems
- [ ] Obtain API keys for existing tools/platforms
- [ ] Set up secure credential sharing (1Password/LastPass)
- [ ] Configure VPN access if required
- [ ] Test all system access and permissions

### Documentation & Resources
- [ ] Collect current process documentation
- [ ] Receive organizational chart and key stakeholder list
- [ ] Obtain existing workflow diagrams/maps
- [ ] Review company policies and compliance requirements
- [ ] Gather previous automation attempts/tools used

### Legal & Administrative
- [ ] Execute signed service agreement
- [ ] Complete data processing agreement (DPA)
- [ ] Finalize project scope and deliverables
- [ ] Set up project billing and invoicing
- [ ] Add client contacts to project management system
